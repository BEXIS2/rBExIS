# rBExIS

An R package to exchange data and metadata in between R and BExIS II.

## Install

```
# install devtools
install.packages("devtools")
# install rBExIS package from github
library(devtools)
install_github("cpfaff/rBExIS", subdir = "rBExIS")
```
